# Ondeón Campaign Hub

Dashboard de gestión de campañas de marketing digital para hilo musical libre de SGAE.

## Estructura
- `index.html` — Punto de entrada
- `assets/` — JS y CSS compilados

## Despliegue
Sitio estático listo para servir. Configurado para desplegarse en `hub.ondeon.es` vía IONOS Deploy Now.

## Tecnología
- React + TypeScript
- Tailwind CSS + shadcn/ui
- Recharts para visualización de datos
- General Sans (Fontshare)
